## 為什麼

Airtype 處理敏感資料（語音、文字、API 金鑰）。安全性審查確保資料隱私承諾得以實現：音訊永不持久化、API 金鑰已加密、設定權限正確，且日誌不洩漏內容。

參考：PRD §9（安全性與隱私）。

相依性：所有前序變更。

## 變更內容

- 驗證音訊資料永不寫入磁碟
- 透過系統 keyring 實作 API 金鑰加密
- 驗證設定目錄權限（0o700）
- 審查日誌輸出以確保不記錄辨識文字
- 產生安全性審查報告
